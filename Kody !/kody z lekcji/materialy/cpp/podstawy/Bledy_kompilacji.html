Błędy kompilacji

Created czwartek 18 wrzesień 2014

Słownik błędów

  • error - błąd
  • warning - ostrzeżenie
  • variable - zmienna
  • (not) set - (nie) ustawiona, (nie) zadeklarowana
  • (not) used - (nie) użyta
  • uinitialized - niezainicjowana

Typowe błędy

1) Brak dołączonych bibliotek na początku programu:

  • #include <iostream> //biblioteka wejścia - wyjścia
  • #include <cstdlib> //standardowa biblioteka c
  • #include <math.h> //bilioteka funkcji i zmiennych matematycznych

2) Brak funkcji main:
int main() {

return 0;
}

3) Niedomknięte nawiasy i cudzysłowy:

  • w blokach {}
  • w wyrażeniach ()
  • w napisach "napis"

4) Błędy typograficzne w nazwach funkcji, niewłasciwe operatory np.:

  • ciout >> zamiast cout << "napis"
  • ciin << zamiast cin >> zmienna

5) Brak średnika konczącego polecenie.
